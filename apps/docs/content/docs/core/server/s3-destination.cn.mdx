S3 目标部分对于备份数据库至关重要。

## 操作：

1. **创建**: 创建一个新的 S3 目标。
2. **删除**: 删除一个 S3 目标。
3. **编辑**: 编辑一个 S3 目标。
4. **测试**: 测试一个 S3 目标。

### 创建

要创建一个新的 S3 桶，您需要填写表单。

- **名称**: 这可以是您想要的任何名称，它将是桶的名称。
- **访问密钥**: 这是您用来访问桶的访问密钥。
- **秘密密钥**: 这是您用来访问桶的秘密密钥。
- **桶**: 这是您将用来访问桶的桶名称。
- **区域**: 这是您将用来访问桶的区域。
- **端点**: 这是您将用来访问桶的端点。

<Callout>
 有一个按钮 `测试`，用于测试与您的桶的连接，如果正确，它会显示成功消息。
</Callout>
